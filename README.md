# 퇴근 후 병원 찾기 (Night Opeing) - Toy Project

퇴근 후에 급하게 병원을 가야하는 일이 생겼을 때, 지금 진료중인 병원만 보고 싶을 때 현재 나의 위치에서 제일 가까운 진료중인 병원을 찾아주는 서비스

## 프로젝트 소개

개발 기간 : 2024년 8월 4일 ~

## 시작 가이드

- Node.js 18.20.4

```
yarn
yarn dev
```
